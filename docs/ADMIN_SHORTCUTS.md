# Admin Panel - Keyboard Shortcuts

## Navigation

| Shortcut | Action | Context |
|----------|--------|---------|
| `/` | Focus search input | When search input is available |
| `Esc` | Close dialog/modal | When dialog is open |

## Quick Actions

| Shortcut | Action | Context |
|----------|--------|---------|
| `Cmd+K` (Mac) / `Ctrl+K` (Windows) | Command palette | Anywhere (planned) |
| `Cmd+U` (Mac) / `Ctrl+U` (Windows) | Navigate to upload | Anywhere (planned) |

## Document Management

| Shortcut | Action | Context |
|----------|--------|---------|
| `Enter` | Open document details | When document is focused |
| `Delete` | Delete document | When document is selected (planned) |

## Search Testing

| Shortcut | Action | Context |
|----------|--------|---------|
| `Enter` | Execute search | When in search query input |
| `Cmd+Enter` | Execute with debug mode | When in search query input (planned) |

## Settings

| Shortcut | Action | Context |
|----------|--------|---------|
| `Cmd+S` (Mac) / `Ctrl+S` (Windows) | Save settings | When in Settings page (planned) |

## Tips

- Press `?` to show keyboard shortcuts overlay (planned)
- Most shortcuts follow standard web conventions
- Additional shortcuts will be added in future updates
